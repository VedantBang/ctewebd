Welcome to CTE WebD - Backend Practice

This set of pages contains various questions to practice your backend skills. The questions are based on ExpressJS.

Important Points:

  • Each page listed below contains a question to solve.
  • You need to code a backend server to fulfill the requirements given in question, and host it on your own machine (localhost).
  • Once you have hosted the server on your machine, you need to make it available on the internet using Ngrok.
  • Each question has a textfield input, where you need to input the ngrok hosted address of your server.
  • On pressing the submit button, the code built into the page will evaluate if your server meets the requirements of the question.
  • The page will make requests to your server to evaluate it, and then display the results on your screen.
  • At the end of the question is a checklist of the objectives. When you submit your address, the objectives in the checklist will change their color.
  • On sucessful completion, an objective turns green, and on an error, the objective turns red.
  • Make sure to turn off your Ngrok connection at the end.
